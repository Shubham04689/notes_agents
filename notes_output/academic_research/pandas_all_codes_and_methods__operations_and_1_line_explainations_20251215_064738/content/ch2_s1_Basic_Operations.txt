3. Detailed Explanations with Examples

   a. Selection
      Selection in Pandas allows for the extraction of specific rows and/or columns based on labels or positions. We can utilize the `loc` and `iloc` functions for this purpose.

         Example:
         ```python
         import pandas as pd
         data = {'Name': ['Tom', 'Nick', 'John'], 'Age': [20, 21, 19]}
         df = pd.DataFrame(data)
         print(df.loc[1, 'Age'])  # prints 21
         print(df.iloc[1, 1])  # prints 21
         ```

   b. Filtering
      Filtering DataFrames is accomplished by applying conditions to specific columns.

         Example:
         ```python
         print(df[df['Age'] > 20])  # prints rows where Age > 20
         ```

   c. Sorting
      Sorting in Pandas can be achieved by utilizing the `sort_values` function.

         Example:
         ```python
         df.sort_values('Age', inplace=True)  # sorts DataFrame by Age in ascending order
         ```

   d. Modifying Data
      Modifying data involves replacing values in specific columns with new values.

         Example:
         ```python
         df['Name'] = ['Tommy', 'Nicky', 'Johnny']  # replaces Name column values
         print(df)
         ```

   e. Merging and Joining DataFrames
      The `merge` function in Pandas is used to combine two DataFrames based on a common column.

         Example:
         ```python
         df1 = pd.DataFrame({'Name': ['Tom', 'Nick'], 'Age1': [20, 21]})
         df2 = pd.DataFrame({'Name': ['Tom', 'Jim'], 'Age2': [22, 24]})
         print(pd.merge(df1, df2, on='Name'))  # merges DataFrames based on Name column
         ```
