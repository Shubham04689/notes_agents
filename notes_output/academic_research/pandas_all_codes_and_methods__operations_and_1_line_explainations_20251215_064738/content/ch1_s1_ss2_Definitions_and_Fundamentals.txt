Definitions and Core Concepts

1.1 Introduction and Definitions

Pandas, an essential extension to Python's numerical and scientific capabilities, provides data structures and functions for manipulating structured data. The primary data structures in Pandas are Series and DataFrame.

- Series: A one-dimensional labeled array, capable of holding any data type.
- DataFrame: A two-dimensional labeled data structure, characterized by rows and columns.

1.2 Core Concepts and Principles

1.2.1 Labels: Facilitate indexing of both rows and columns for efficient data access.
1.2.2 Alignment: Ensures data integrity by aligning data based on labels.
1.2.3 Grouping: Aggregates and analyzes data based on labels for meaningful insights.
1.2.4 Reshaping: Adjusts data between Series, DataFrame, and higher-dimensional structures using methods like melt() and pivot_table().

Detailed Explanations with Examples

1.3.1 Creating a DataFrame:

```python
import pandas as pd
data = {'Name': ['John', 'Anna', 'Peter'],
        'Age': [28, 24, 35],
        'City': ['New York', 'Los Angeles', 'Chicago']}
df = pd.DataFrame(data)
```

1.3.2 Accessing Data:

```python
print(df['Name'])  # Accessing column
print(df.loc[1])   # Accessing row
print(df.iloc[1])  # Accessing row using integer index
```
