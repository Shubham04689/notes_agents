3. Practical Application and Illustrations

   a. Data Manipulation:

      i. Grouping Data: Group data based on one or more columns using the `groupby()` function.

            Example: Group sales data by year and calculate the total sales for each group.

            ```python
            sales_grouped = sales_data.groupby('Year')['Sales'].sum()
            ```

      ii. Reshaping Data: Transform data from wide to long or vice versa using the `melt()` and `pivot_table()` functions.

            Example: Reshape sales data to show sales for each product across years.

            ```python
            sales_melted = pd.melt(sales_data, id_vars=['Year', 'Product'], var_name='Month', value_name='Sales')
            ```

   b. Meaningful Data Representations:

      i. Heatmaps: Visualize the correlation between variables using a matrix of colors.

            Example: Create a heatmap to show the correlation between various financial indicators.

            ```python
            corr_matrix = sales_data[['Revenue', 'Costs', 'Profit']].corr()
            sns.heatmap(corr_matrix)
            ```

      ii. Bar Plots with Error Bars: Visualize mean values and the dispersion of values around the mean using error bars.

            Example: Create a bar plot with error bars to show the average sales for each region and the standard deviation.

            ```python
            sales_region = sales_data.groupby('Region')['Sales'].mean()
            sales_region_sd = sales_data.groupby('Region')['Sales'].std()
            plt.bar(sales_region.index, sales_region, yerr=sales_region_sd)
            ```
