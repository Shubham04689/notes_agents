**Chapter 3: Hands-On Learning**

3.1 Creating and Accessing DataFrames

Let's dive into the practical applications of Pandas by learning how to create a DataFrame and access data within it.

```python
import pandas as pd

data = {
    'Name': ['John', 'Anna', 'Peter'],
    'Age': [28, 24, 35],
    'City': ['New York', 'Los Angeles', 'Chicago']
}

df = pd.DataFrame(data)
print(df)
```

3.2 Navigating DataFrames

```python
print(df['Name'])  # Access column
print(df.iloc[1])  # Access row
print(df.loc[1, 'Name'])  # Access specific cell
```
