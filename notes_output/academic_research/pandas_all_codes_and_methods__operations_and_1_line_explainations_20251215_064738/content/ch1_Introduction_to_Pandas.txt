Chapter 1: Introduction to Pandas

1.1 Understanding Pandas

Pandas is a powerful data analysis and manipulation library in Python, providing users with essential tools for managing data effectively. With its core functions, users can handle data structures such as Series, DataFrame, and Panel, making it an indispensable asset in the data science world.

1.2 Core Concepts and Principles

1.2.1 DataFrame

At the heart of Pandas lies the DataFrame, a table-like data structure with rows (indexed) and columns (labeled). It serves as the primary data structure, offering high flexibility for data manipulation.

1.2.2 Series

A Series is a one-dimensional labeled array-like object, acting as a single column in a DataFrame.

1.2.3 Indexing

Both DataFrames and Series are indexed, allowing for efficient access and manipulation of data.

1.2.4 Data Alignment

Pandas automatically aligns data during operations, ensuring consistent indices for seamless data integration.

1.3 Detailed Explanations with Examples

1.3.1 Creating a DataFrame

```python
import pandas as pd
data = {'Name': ['John', 'Anna', 'Peter'], 'Age': [28, 24, 35]}
df = pd.DataFrame(data)
print(df)
```
Output:
```
   Name  Age
0  John   28
1  Anna   24
2  Peter  35
```

1.3.2 Accessing Data

```python
print(df['Name'][1])  # Accessing Anna's name
print(df.loc[1, 'Age'])  # Accessing Anna's age
```

1.3.3 Data Manipulation

```python
df['New_Column'] = [1, 2, 3]  # Adding a new column
df.drop('Name', axis=1)  # Dropping the Name column
```

1.3.4 Filtering Data

```python
df[df['Age'] > 30]  # Filtering rows where Age is greater than 30
```
