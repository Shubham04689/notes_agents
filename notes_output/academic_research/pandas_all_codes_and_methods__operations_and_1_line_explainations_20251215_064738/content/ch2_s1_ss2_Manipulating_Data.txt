3. Practical Exercises: Detailed Explanations with Examples

a. DataFrame Operations

i. DataFrame Assignment: Assign a new column to an existing DataFrame using the `[]` operator.

Example:
```python
import pandas as pd
data = {'Name': ['John', 'Anna', 'Peter'], 'Age': [28, 24, 35]}
df = pd.DataFrame(data)
df['City'] = ['New York', 'Los Angeles', 'Chicago']
print(df)
```

ii. DataFrame Merge: Merge two DataFrames based on a common column using the `merge()` function.

Example:
```python
df1 = pd.DataFrame({'ID': [1, 2, 3], 'Score1': [85, 90, 70]})
df2 = pd.DataFrame({'ID': [1, 2, 3], 'Score2': [90, 80, 85]})
merged_df = pd.merge(df1, df2, on='ID')
print(merged_df)
```

b. DataFrame Grouping and Aggregation

i. Group By: Group data based on a column and perform operations on each group using the `groupby()` function.

Example:
```python
data = {'Name': ['John', 'Anna', 'Peter', 'John', 'Anna'], 'Age': [28, 24, 35, 28, 24]}
df = pd.DataFrame(data)
grouped_df = df.groupby('Name').agg({'Age': 'mean'})
print(grouped_df)
```
